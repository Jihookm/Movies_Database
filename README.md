# Movies_Database
2020-2학기 데이터베이스 설계 프로젝트 <br><br>

MOVIES ERD를 기반으로 MOVIES 데이터베이스를 설계합니다.<br>
이를 MySQL을 통해 구현한 후 Java를 이용하여 web application을 구현합니다.
